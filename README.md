# This repo conataines a qr code generator that is made with Python + qrcode module.

----------------

## How to install and generate :

### On Linux or MacOs : (Run these command in the terminal)

- git clone https://github.com/frginlv/QR-Code-generator-.git
- cd QR-Code-generator-/'qr codes generator'
- python3 main.py

### On Windows : (Run these command in the terminal)

- git clone https://github.com/frginlv/QR-Code-generator-.git
- cd QR-Code-generator-/'qr codes generator'
- python main.py

## After runing these commands , check the same directory to find the qrcode as a png file.
